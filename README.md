# LangGraph Tutorial Series

A comprehensive tutorial series for learning LangGraph from basics to advanced patterns.

## Setup

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Configure environment variables:**
   ```bash
   cp .env.example .env
   # Edit .env with your Azure OpenAI credentials
   ```

3. **Run tutorials:**
   Start with `01_hello_world.py` and work your way through sequentially:
   ```bash
   python 01_hello_world.py
   python 02_state_management.py
   # ... and so on
   ```

## Tutorial Structure

- **Level 1 (01-05)**: Fundamentals - Basic concepts and state management
- **Level 2 (06-10)**: Intermediate - Error handling, persistence, streaming
- **Level 3 (11-15)**: Advanced - Agent patterns and complex workflows
- **Level 4 (16-20)**: Expert - Production patterns and real-world applications

See `TUTORIAL_PLAN.md` for detailed information about each tutorial.

## Prerequisites

- Python 3.8+
- Azure OpenAI account and API credentials
- Basic understanding of Python
- Familiarity with LangChain (covered in separate repo)

## Learning Path

Complete tutorials in order as each builds upon previous concepts.
